python个人笔记1

#!/usr/bin/env python3     或     #!/usr/bin/python3
#coding=utf-8
用来指定由哪个解释器来执行脚本,和指定字符编码

python中的关键字:

在python中TAB缩进很重要，缩进不一样可能会导致错误

python中如果语句很长可以用反斜杠(\)来实现多行一句，但是在{},[],()中是不需要用\来表示换行的

在python的输出中：

print( 变量[头下标:尾下标:步长]     sequence[start : end : step] )

输出为:

step:表示切片，就是从几开始数到第几个,中间的不要了，如果是负数就把整个字符串倒序

在输出的前面加r可以使转义字符\和后面的字符显示

print默认输出换行，如果要输出不换行需要在变量末尾加上end=”  ”

python导入模块：

import格式：
整个模块导入:import somemodule
from...import格式:
导入某个模块中的某个函数:from somemodule import somefunction
导入某个模块中的多个函数:from somemodule import first,second
导入整个模块:from somemodule import *

python的标准数据类型:

不可变数据:
Number(数字)       支持  int,float,bool,complex(复数)

可以用isinstance(object,classinfo)来判断变量的类型

isinstance和type的区别在于isinstace对认为子类是一种父类类型，而type不会
在isinstance中如果classinfo不是一种数据类型或者由数据类型构成的元组，将引发一个TypeError异常。

String(字符串)

加号 + 是列表连接运算符，星号 * 是重复操作

Tuple(元组)
可变数据:
List(列表)
Set(集合)
Dictionary(字典)

var是动态变量名，是可以不需要预先定义数据类型的，即使两个变量名相同，但是两个变量的ID是不同的，所以可以使用相同的名字来定义不同值的变量。

可以使用del来删除对象引用

输出为:

Tuple(元组),String和List都属于 sequence(序列)

Tuple的0个元素和1个元素是比较特殊:

set(集合)的基本功能是进行成员关系测试和删除重复元素
     可以使用{}和set()函数创建集合
     创建一个空集的时候要用set(),因为{}是字典的空集

输出为:

dictionary(字典)
d = {key1 : value1,key2 : value2},一个键对应一个键值(key : value)，这些键是没有先后顺序的,可以直接添加，更改和删除:

python的数据类型的转化:
int(x,[base]) 将x转化为一个整数 (默认base=10，当base是其他值时转化为其他进制)
hex(x) 将一个整数转化为一个十六进制字符串
oct(x) 将一个整数转化为一个八进制字符串
str(x) 将对象转化为字符串
ord(x) 将一个转化为它的整数值
chr(x) 将一个整数转化为一个字符
frozenset(x) 转化为不可变集合
