Number的内建函数

 

数学函数:

函数 函数描述 函数示例
abs(x) 绝对值函数 abs(-10)返回10
ceil(x) 向上取整 math.ceil(4.1)返回5
exp(x) 返回e的x次幂(ex) math.exp(1)返回2.718281828459045
fabs(x) 返回数字的绝对值 math.fabs(-10)返回10.0
floor(x) 向下取整 math.floor(4.9)返回4
log(x) log函数 math.log(100,10)返回2.0
log10(x) log10函数 math.log10(100)返回2.0
max(x1, x2,...) 返回最大值 max(80, 100, 1000)返回1000
min(x1, x2,...) 返回最小值 min(80, 100, 1000)返回80
modf(x) 返回x的整数部分与小数部分，两部分的数值符号与x相同，整数部分以浮点型表示 math.modf(100.12)返回(0.12000000000000455, 100.0)
pow(x, y) x**y 运算后的值 math.pow(100, 2)返回10000.0
round(x [,n]) 返回浮点数x的四舍五入值，如给出n值，则代表舍入到小数点后的位数 round(-100.000056, 3)返回-100.0
sqrt(x) 返回数字x的平方根 math.sqrt(7)返回2.64575131106

三角函数

函数 函数描述 函数示例
acos(x) 返回x的反余弦弧度值 math.acos(0.64)返回0.876298061168
asin(x) 返回x的反正弦弧度值 math.asin(0.64)返回0.694498265627
atan(x) 返回x的反正切弧度值 math.atan(0.64)返回0.569313191101
atan2(y, x) 返回给定的 X 及 Y 坐标值的反正切值 math.atan2(-0.50,-0.50)返回-2.35619449019
cos(x) 返回x的弧度的余弦值 math.cos(3)返回-0.9899924966
hypot(x, y) 返回欧几里德范数 sqrt(x*x + y*y) math.hypot(3, 2)返回3.60555127546
sin(x) 返回的x弧度的正弦值 math.sin(3)返回0.14112000806
tan(x) 返回x弧度的正切值 math.tan(3)返回-0.142546543074
degrees(x) 将弧度转换为角度 math.degrees(3)返回171.887338539
radians(x) 将角度转换为弧度 math.radians(3)返回0.0523598775598

随机数函数

函数 函数描述 函数示例
choice(seq) 从序列的元素中随机挑选一个元素 random.choice(range(10))返回0
randrange ([start,] stop [,step]) 从指定范围内，按指定基数递增的集合中获取一个随机数，基数默认值为 1 random.randrange(1, 100, 2)返回16
random() 随机生成下一个实数，它在[0,1)范围内 random.random()返回0.281954791393
seed([x]) 改变随机数生成器的种子seed。如果你不了解其原理，你不必特别去设定seed，Python会帮你选择seed,(就是相当于游戏中Maps的种子) 下方的图片为示例
shuffle(lst) 将序列的所有元素随机排序 list = [20, 16, 10, 5]     random.shuffle(list)返回[16, 5, 10, 20]
uniform(x, y) 随机生成下一个实数，它在[x,y]范围内 random.uniform(5, 10)返回6.98774810047
