    通常使用var来声明变量
    直接赋值的是数字变量
    " " 和 ' '赋值的是文本变量(就差不多字符串那种)
    变量命名规则(直接驼峰就好了):

    变量必须以字母开头
    变量也能以 $ 和 _ 符号开头（不推荐这么做）
    变量名称对大小写敏感 (y 和 Y 是不同的变量)

  1. 事件: (发生在 HTML 元素上的事情)

    常见的事件:

    事件 描述
    onchange HTML 元素改变
    onclick 用户点击 HTML 元素
    onmouseover 用户在一个HTML元素上移动鼠标

    onkeydown 用户按下键盘按键
    onload 浏览器已完成页面的加载

属性 描述

length 返回字符串的长度
prototype 允许您向对象添加属性和方法(Prototype 是全局属性，适用于所有的 Javascript 对象)

  1. 比较运算符中有两个比较特殊的:(按照x=5为例)
    运算符 描述 比较 返回值
    == 等于 x==8 false
    x==5 true
    === 绝对等于(值和类型均相等) x==="5"(这是字符串) false
    x===5 true
    != 不等于 x!=8 true
    !== 不绝对等于(值和类型有一个不相等，或两个都不相等) x!==5 false
    x!=="5" true

条件运算符: variablename=(condition)?value1:value2

for/in 语句循环遍历对象的语法:for (x in person)

  1. Break 和 Continue 的标签:

    continue 语句（带有或不带标签引用）只能用在循环中
    break 语句（不带标签引用），只能用在循环或 switch 中
    通过标签引用，break 语句可用于跳出任何 JavaScript 代码块：    (语法:   label:statements )

  2. typeof, null, 和 undefined:

    typeof操作符来检测变量的数据类型      typeof 3.14(返回 number)
    null为空，返回的typeof为object
    undefined返回的typeof为undefined
    undefined    表示变量声明过但并未赋过值
    null               表示一个变量将来可能指向一个对象。

  3. 类型转换:

    数字转化为字符串:

    方法 描述
    toString() 把对象的值转化为字符串
    toExponential() 把对象的值转换为指数计数法
    toFixed() 把数字转换为字符串，结果的小数点后有指定位数的数字
    toPrecision() 把数字格式化为指定的长度

得到日期:

方法 描述
getDate() 从 Date 对象返回一个月中的某一天 (1 ~ 31)
getDay() 从 Date 对象返回一周中的某一天 (0 ~ 6)
getFullYear() 从 Date 对象以四位数字返回年份
getHours() 返回 Date 对象的小时 (0 ~ 23)
getMilliseconds() 返回 Date 对象的毫秒(0 ~ 999)
getMinutes() 返回 Date 对象的分钟 (0 ~ 59)
getMonth() 从 Date 对象返回月份 (0 ~ 11)
getSeconds() 返回 Date 对象的秒数 (0 ~ 59)
getTime() 返回 1970 年 1 月 1 日至今的毫秒数

将字符串转换为数字:

方法 描述
parseFloat() 解析一个字符串，并返回一个浮点数
parseInt() 解析一个字符串，并返回一个整数
  1. 正则表达式

    语法:/正则表达式主体/修饰符(可选)
    正则表达式通常用于两个字符串方法:

    search() 测试字符串里面有没有匹配的子串，有的话返回子串的起始位置
    replace()替换字符串里面的字串

    使用 RegExp 对象:

    test()测试字符串里面有没有匹配的，有返回true，没有返回false
    exec()测试正则表达式中有没有匹配的结果，有的话返回这个值，没有返回null

    正则表达式修饰符：

    修饰符 描述
    i 执行对大小写不敏感的匹配
    g 执行全局匹配(查找所有匹配而非在找到第一个匹配后停止)
    m 执行全局匹配多行匹配

正则表达式表达式：

表达式 描述
[abc] 查找方括号之间的任何字符
[^abc] 查找任何不在方括号之间的字符
[0-9] 查找任何从 0 至 9 的数字
[a-z] 查找任何从小写 a 到小写 z 的字符
[A-Z] 查找任何从大写 A 到大写 Z 的字符
[A-z] 查找任何从大写 A 到小写 z 的字符
(x|y) 查找任何以 | 分隔的选项。

正则表达式元字符：(小写代表查找的那个，大写代表查找非的那个)

元字符 描述
\w 查找单词字符(就是不算符号和空格)
\W 查找非单词字符
\d 查找数字
\s 查找空白字符
\b 匹配单词边界 或者 查找位于单词的开头或结尾的匹配(如果查找目标是单词中间则返回null)
\uxxxx 查找以十六进制数 xxxx 规定的 Unicode 字符

正则表达式量词:

量词 描述
n+ 匹配任何包含至少一个 n 的字符串
n* 匹配任何包含零个或多个 n 的字符串
n? 匹配任何包含零个或一个 n 的字符串

    finally 语句在 try 和 catch 语句之后，无论是否有触发异常，该语句都会执行

  1. 变量提升:

    变量的定义会被全部前置到函数的最顶部
    变量的赋值不会被前置
    提升（Hoisting）是 JavaScript 默认将当前作用域提升到前面去的的行为

  2. 严格模式：

    这个模式就相当于变成了标准的语法格式，需要定义才能赋值，变量不能重名，不能删除等
    在最前面加上"use strict"就可以变成严格模式

  3. let 和 const：

    let声明的变量只在 let 命令所在的代码块内有效
    const声明一个只读的常量，一旦声明，常量的值就不能改变
