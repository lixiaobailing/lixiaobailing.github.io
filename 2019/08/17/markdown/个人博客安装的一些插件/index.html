个人博客安装的一些插件

lixiaobailing

2019-08-16

HEXO配置

  1. live-2d的插件(可爱miku的插件)：

      npm install --save hexo-helper-live2d
                   (安装live-2d)
      npm install live2d-widget-model-miku (你需要的人物的名字)
                   (安装可爱的miku)

  2. widget: related_posts的插件(一些文献的插件)：

      npm i -S hexo-related-popular-posts
                   (安装此插件)

  3. 安装二维码的插件：

      npm i -S hexo-helper-qrcode
                   (安装此插件)

  4. 安装git上传：

      npm install hexo-deployer-git --save
                   (安装此插件)
      修改根目录下的”_config.yml”设置的deploy里面的type，把github改为git
      hexo deploy(在终端里面)
                  

  5. 文章字数+阅读时长统计：

      npm i –save hexo-wordcount
                   (安装此插件)

  6. hexo文章加密码：

      npm install --save hexo-blog-encrypt
                   (安装此插件)
      修改根目录下的”_config.yml”设置的Security里面的encrypt，把false改为true
      在文章的开头的选项里面加入password, abstract, message这三项设置

  7. 图片无法显示:

      

  8. 百度链接主动推送:

      npm install hexo-baidu-url-submit –save
                   (安装此插件)

  9. 文章添加 emoji 表情符:

      npm install hexo-filter-github-emojis --save
                   (安装此插件)
      在根目录下的”_config.yml”下的一些设置：(具体的一些设置参照 hexo-filter-github-emojis)

  10. 站内搜索功能插件：

      npm install --save hexo-generator-searchdb
                   (安装此插件)

  11. 生成RSS文件：

      npm install --save hexo-generator-feed
                   (安装此插件)
      修改根目录下的”_config.yml”设置的Extensions里面的Plugins，添加plugins: hexo-generate-feed
      在主题目录下的”_config.yml”中添加rss: /atom.xml
